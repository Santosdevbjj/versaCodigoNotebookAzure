# Guia de Nomenclatura no Azure

## Padrões
- Prefixos: rg-, dbx-, cl-, st-, kv-
- Ambiente: dev, test, prod
- Região: br (Brazil South), us (East US)

## Exemplos
- Resource Group: rg-dbx-mlops-dev-br
- Workspace: dbx-ws-mlops-dev-br
- Cluster: cl-dbx-dev-br
- Storage: stmlopsdevbr
- Key Vault: kv-mlops-dev-br
